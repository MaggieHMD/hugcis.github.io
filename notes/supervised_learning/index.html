Supervised learning

tags
Machine learning

Data

Input/output example pairs: \[ \{(y_i, x_i)\}_{i\leq n} \sim_{iid} \mathbb{P}, \quad \mathbb{P} \in \mathcal{P}(\mathbb{R} \times \mathbb{R}^d) \text{ unknown} \]

Mapping

We search for a mapping \(f: \mathbb{R}^d \rightarrow \mathbb{R}\)

Objective

The goal is to find the above mapping such as to minimize an objective. For example, the objective can be the quadratic loss below: \[ R(f) = \mathbb{E}\left\{ (y_{new} - f(x_{new}))^2 \right\}, \quad (y_{new}, x_{new}) \sim \mathbb{P} \] Usually, because this objective cannot be minimized directly — we don’t have access to infinitely many new samples — minimization on the set of available examples is done instead: this is Empirical risk minimization (ERM).

For the above \(R\), ERM is written \[ \text{minimize}\quad \hat{R}_n(\theta) := \frac{1}{n}\sum_{i=1}^n(y_i - f(x_i;\theta))^2, \] \[ \text{subj. to } f(\cdot;\theta) : \mathbb{R}^d \rightarrow \mathbb{R}, \quad \theta \in \Theta \subseteq \mathbb{R}^p \]

This minimization problem can be solved using Optimization methods. Gradient descent is one of them. Its stochastic variant is written as follows for the problem above.

\[ \theta^{k+1} = \theta^k - \frac{\epsilon}{2} \nabla_\theta(y_{I(k)} - f(x_{I(k)}; \theta_k))^2 \] \[ I(k) \sim \text{Unif}([n]) \]
